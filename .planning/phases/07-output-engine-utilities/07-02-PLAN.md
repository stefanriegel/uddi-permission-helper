**js/output.js additions** (add to existing module, do not rewrite):

1. **`getActiveTabContent()`** — Returns the text content of the currently visible output panel:
   - Find the panel that is NOT hidden: `document.querySelector('.output__panel:not([hidden])')`
   - Return its `textContent` (strips HTML tags, giving raw policy/terraform/guide text)
   - If no visible panel or content is the placeholder, return `null`

2. **`getActiveTabId()`** — Returns the ID suffix of the active tab:
   - Find `document.querySelector('.output__tab--active')`
   - Return the `aria-controls` value (e.g., `'panel-policy'`, `'panel-terraform'`, `'panel-guide'`)

3. **`getDownloadFilename(providerId, tabId)`** — Returns appropriate filename:
   - Map of tab to extension per context decision:
     - `panel-policy`: AWS -> `aws-policy.json`, Azure -> `azure-policy.sh`, GCP -> `gcp-policy.sh`
     - `panel-terraform`: `{provider}-terraform.tf`
     - `panel-guide`: `{provider}-setup-guide.txt`
   - Return the constructed filename string

4. **`setButtonsDisabled(disabled)`** — Enable/disable copy and download buttons:
   - Select both `[data-action="copy"]` and `[data-action="download"]` buttons
   - Set `disabled` attribute when `disabled=true`, remove when `disabled=false`
   - Add/remove `output__action--disabled` CSS class
   - Call this from `renderOutput()`: disable when selectedIds.length === 0, enable otherwise

Export: `getActiveTabContent`, `getDownloadFilename`, `getActiveTabId`, `setButtonsDisabled`

**js/app.js additions** (add to DOMContentLoaded block):

1. **Copy button handler:**
   ```javascript
   const copyBtn = document.querySelector('[data-action="copy"]');
   if (copyBtn) {
     copyBtn.addEventListener('click', async () => {
       const content = getActiveTabContent();
       if (!content) return;

       try {
         await navigator.clipboard.writeText(content);
       } catch {
         // Fallback for HTTP localhost
         const textarea = document.createElement('textarea');
         textarea.value = content;
         textarea.style.position = 'fixed';
         textarea.style.opacity = '0';
         document.body.appendChild(textarea);
         textarea.select();
         document.execCommand('copy');
         document.body.removeChild(textarea);
       }

       // Confirmation UX: text changes to "Copied!" for 2s with green flash
       const originalText = copyBtn.textContent;
       copyBtn.textContent = 'Copied!';
       copyBtn.classList.add('output__action--copied');
       setTimeout(() => {
         copyBtn.textContent = originalText;
         copyBtn.classList.remove('output__action--copied');
       }, 2000);
     });
   }
   ```

2. **Download button handler:**
   ```javascript
   const downloadBtn = document.querySelector('[data-action="download"]');
   if (downloadBtn) {
     downloadBtn.addEventListener('click', () => {
       const content = getActiveTabContent();
       if (!content) return;

       const providerId = getActiveProvider();
       const tabId = getActiveTabId();
       const filename = getDownloadFilename(providerId, tabId);

       const blob = new Blob([content], { type: 'text/plain;charset=utf-8' });
       const url = URL.createObjectURL(blob);
       const a = document.createElement('a');
       a.href = url;
       a.download = filename;
       document.body.appendChild(a);
       a.click();
       document.body.removeChild(a);
       URL.revokeObjectURL(url);  // Prevent memory leak per CLAUDE.md
     });
   }
   ```

3. Add import of new exports from output.js:
